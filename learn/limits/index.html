小本本有哪些限制？

它只支持 Windows 10/11 x64，需要本机 League Client 提供名单；本地记录不会跨电脑同步，也不能保证系统通知覆盖独占全屏。

更新日期: 2026-07-11

系统与客户端

当前只支持 Windows x64。未运行 League Client 时不能自动获得真实对局名单。

通知限制

Windows 系统通知可能被专注模式或独占全屏隐藏，因此重要记录建议通过 Win+G 主动查看。

数据判断边界

用户评价是私人主观笔记，不代表事实，也不应用于骚扰、诽谤或公开传播。

常见问题

支持 macOS 吗？

当前不支持。

能跨设备同步吗？

当前不能。

能保证每局都识别所有人吗？

不能；客户端接口在选人、加载和进行中阶段提供的字段可能不同，应用会尽量补全。
