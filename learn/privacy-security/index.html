首页 / 使用指南

玩家记录会上传吗？

不会。小本本没有玩家记录服务器、账号或遥测，Riot ID、对局和评价保存在用户自己的 Windows 电脑。

更新日期: 2026-07-11

本地数据

记录只供当前 Windows 用户查看。运营者无法从服务器查询用户写过什么。

必要网络请求

应用可能访问 Riot Data Dragon 获取公开图片，并通过 Microsoft Store 或 GitHub 检查软件更新。

反作弊边界

应用不读取游戏内存、不注入 DLL、不 Hook DirectX，也不尝试绕过 Vanguard。

常见问题

卸载后还有服务器副本吗？

没有，因为记录从未上传。

别人能搜索我的评价吗？

不能。

怎样删除全部数据？

退出应用后删除 %APPDATA%\小本本。
