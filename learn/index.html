小本本使用与隐私指南

了解小本本是什么、如何记录玩家、Win+G 如何工作，以及它明确不做什么。

更新日期: 2026-07-10

小本本是什么？

小本本是一款 Windows 本地玩家笔记工具，帮助英雄联盟玩家记住真人对局中遇到的人以及自己写下的评价。

→

小本本如何记录玩家？

应用打开后读取本机 League Client 与 Live Client 提供的本局名单，排除自己和机器人，再把稳定玩家标识与用户笔记写入本地文件。

→

如何用 Win+G 查看本局记录？

安装小本本后按 Win+G，在小组件菜单打开“小本本”，即可查看本局九名玩家、夯拉统计和以前写过的评价。

→

玩家记录会上传吗？

不会。小本本没有玩家记录服务器、账号或遥测，Riot ID、对局和评价保存在用户自己的 Windows 电脑。

→

小本本有哪些限制？

它只支持 Windows 10/11 x64，需要本机 League Client 提供名单；本地记录不会跨电脑同步，也不能保证系统通知覆盖独占全屏。
