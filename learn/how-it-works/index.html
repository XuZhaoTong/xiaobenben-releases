小本本如何记录玩家？

应用打开后读取本机 League Client 与 Live Client 提供的本局名单，排除自己和机器人，再把稳定玩家标识与用户笔记写入本地文件。

更新日期: 2026-07-10

数据流程

League Client 提供 Riot ID 和公开玩家标识；Live Client 补充进行中的对局信息；应用合并来源并避免重复建局。

匹配旧记录

优先使用 PUUID、召唤师 ID 和完整 Riot ID 匹配，避免仅凭相同昵称误合并玩家。

保存位置

桌面端记录保存在当前 Windows 用户的应用数据目录，并向同一安装包中的 Game Bar 小组件同步只读状态。

常见问题

客户端没开会怎样？

无法读取真实名单，但应用仍可查看和手动编辑已有记录。

会读取聊天内容吗？

不会。

改名后还能匹配吗？

存在稳定玩家标识时可以；只有昵称的数据可能需要手动确认。
