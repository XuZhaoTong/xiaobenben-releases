如何用 Win+G 查看本局记录？

安装小本本后按 Win+G，在小组件菜单打开“小本本”，即可查看本局九名玩家、夯拉统计和以前写过的评价。

更新日期: 2026-07-11

为什么使用 Game Bar

Game Bar 是 Windows 的游戏覆盖层，不需要向英雄联盟进程注入代码，也不会像普通置顶窗口那样把独占全屏切回桌面。

使用步骤

先启动桌面端让它记录名单；进入对局后按 Win+G；从小组件菜单打开小本本；点玩家查看历史评价。

是否必须固定

不需要。按 Win+G 临时查看即可，关闭 Game Bar 后不会挡住游戏视野。

常见问题

找不到小组件怎么办？

先确认安装包完整安装并在 Game Bar 小组件菜单中查找。

能在里面写评价吗？

当前 Game Bar 版本以查看为主，完整编辑在桌面端完成。

需要重启游戏吗？

通常不需要；首次安装 Game Bar 组件后重新打开 Win+G 即可。
