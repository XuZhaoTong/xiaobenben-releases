能力与边界

它会做

  • 记录真人对局中除自己以外的玩家
  • 保存 Riot ID、对局时间和用户自己的评价
  • 命中已有记录玩家时发送可关闭的开局提醒
  • 将记录同步给本机 Xbox Game Bar 小组件

它不会做

  • 不会分析胜率、水平或预测胜负
  • 不会建立公开玩家评分或上传评价
  • 不会读取游戏内存、注入进程或绕过反作弊
  • 不会记录人机对局

常见问题指南

围绕安装、工作原理、隐私和兼容性的直接答案。

小本本是什么？

小本本是一款 Windows 本地玩家笔记工具，帮助英雄联盟玩家记住真人对局中遇到的人以及自己写下的评价。

→

小本本如何记录玩家？

应用打开后读取本机 League Client 与 Live Client 提供的本局名单，排除自己和机器人，再把稳定玩家标识与用户笔记写入本地文件。

→

如何用 Win+G 查看本局记录？

安装小本本后按 Win+G，在小组件菜单打开“小本本”，即可查看本局九名玩家、夯拉统计和以前写过的评价。

→

玩家记录会上传吗？

不会。小本本没有玩家记录服务器、账号或遥测，Riot ID、对局和评价保存在用户自己的 Windows 电脑。

→

小本本有哪些限制？

它只支持 Windows 10/11 x64，需要本机 League Client 提供名单；本地记录不会跨电脑同步，也不能保证系统通知覆盖独占全屏。

→

常见问题

小本本需要 Riot 账号吗？

不需要。它只读取本机已经登录的 League Client 提供的本局名单。

我的评价会被其他玩家看到吗？

不会。评价只保存在当前电脑，不会上传，也没有公开查询页。

会不会影响 Vanguard？

小本本不注入游戏、不读取游戏内存，只访问本机客户端接口和自己的本地文件。

人机局会记录吗？

不会。检测到机器人玩家后，整局不会写入对局记录。
