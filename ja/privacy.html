プライバシーと免責事項

XiaoBenBen のデータ処理と Riot Games に関する免責事項。

範囲

Windows アプリ、Xbox Game Bar ウィジェット、公開サイトに適用されます。施行日：2026年7月10日。

データ

ローカル API から Riot ID、公開識別子、チャンピオン、画像を読み、自分の評価、メモ、試合時刻を PC に保存します。

ネットワーク

Riot Data Dragon の公開画像と Microsoft Store または GitHub の更新にアクセスする場合があります。広告やテレメトリはありません。

共有と削除

運営者はメモを収集・販売・共有しません。アプリ終了後に %APPDATA%\小本本 を削除すると全データを消去できます。

セキュリティ

ゲームメモリを読まず、コードを注入せず、Riot のパスワードを要求しません。

サポート

GitHub Issues から問い合わせできます。 GitHub Issues

Riot Games 免責

XiaoBenBen は Riot Games の承認製品ではありません。関連する商標は Riot Games, Inc. に帰属します。
