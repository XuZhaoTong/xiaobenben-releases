ローカル保存の League of Legends プレイヤーメモ

XiaoBenBen

対人戦で出会ったプレイヤーを自動記録し、良い・避ける評価と個人メモを残せます。再会したときは Windows 通知と Win+G で過去の記録を確認できます。

XiaoBenBen の機能

過去の味方や対戦相手についての自分の記憶を PC に保存します。

自動記録

対人戦プレイヤーを保存

起動中にローカルの League Client から名簿を読み、ボット戦は除外します。

Win+G

ゲーム内で過去メモを確認

Xbox Game Bar で現在のプレイヤーと以前のメモを確認できます。

個人メモ

良い・避ける・テキスト

試合後に評価し、手動でプレイヤーを追加・編集できます。

機能と境界

行うこと

  • 対人戦の他プレイヤーを記録
  • Riot ID、試合時刻、自分のメモを保存
  • 記録済みプレイヤーがいる時だけ通知
  • Game Bar に読み取り専用状態を同期

行わないこと

  • 実力や勝率、勝敗の予測
  • 公開評価スコアの作成
  • ゲームメモリの読み取りやコード注入
  • ボット戦の記録

利用ガイド

セットアップ、データ、プライバシー、互換性の説明です。

XiaoBenBen とは？

対人戦で会ったプレイヤーと個人メモを覚えるための Windows ローカルツールです。

→

プレイヤーをどう記録しますか？

ローカル League Client の名簿を読み、自分とボットを除外して識別子とメモを PC に保存します。

→

Win+G でどう確認しますか？

Win+G の XiaoBenBen ウィジェットで現在の名簿と過去メモを確認します。

→

メモはアップロードされますか？

されません。アカウントやメモ用サーバーはなく、データは PC に残ります。

→

制限はありますか？

Windows x64 とローカル League Client が必要で、端末間同期はありません。

→

よくある質問

Riot アカウントは必要ですか？

不要です。PC でログイン済みの League Client が提供する名簿だけを読みます。

他の人にメモが見えますか？

見えません。メモは PC 内に保存されます。

Vanguard に影響しますか？

ゲームへの注入やゲームメモリの読み取りは行いません。

ボット戦も記録しますか？

しません。ボットを検出した試合は全体を除外します。
