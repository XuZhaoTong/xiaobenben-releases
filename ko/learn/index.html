업데이트: 2026-07-10

XiaoBenBen이란?

실제 경기에서 만난 플레이어와 개인 메모를 기억하는 Windows 로컬 도구입니다.

플레이어를 어떻게 기록하나요?

로컬 League Client 명단을 읽고 자신과 봇을 제외한 뒤 식별자와 메모를 PC에 저장합니다.

Win+G로 어떻게 확인하나요?

Win+G에서 XiaoBenBen 위젯을 열어 현재 명단과 과거 메모를 확인합니다.

메모가 업로드되나요?

아니요. 계정이나 메모 서버가 없으며 데이터는 PC에만 남습니다.

어떤 제한이 있나요?

Windows x64와 로컬 League Client가 필요하고 장치 간 동기화는 지원하지 않습니다.
