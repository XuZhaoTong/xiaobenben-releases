기능과 한계

하는 일

  • 실제 경기의 다른 플레이어 기록
  • Riot ID, 경기 시간과 개인 메모 저장
  • 기록한 플레이어를 만났을 때만 알림
  • 로컬 Game Bar 위젯에 읽기 전용 상태 동기화

하지 않는 일

  • 실력·승률·승패 예측
  • 공개 평판 점수 생성
  • 게임 메모리 읽기나 코드 주입
  • 봇 경기 기록

사용 가이드

설정, 데이터 흐름, 개인정보와 호환성에 대한 답변입니다.

XiaoBenBen이란?

실제 경기에서 만난 플레이어와 개인 메모를 기억하는 Windows 로컬 도구입니다.

→

플레이어를 어떻게 기록하나요?

로컬 League Client 명단을 읽고 자신과 봇을 제외한 뒤 식별자와 메모를 PC에 저장합니다.

→

Win+G로 어떻게 확인하나요?

Win+G에서 XiaoBenBen 위젯을 열어 현재 명단과 과거 메모를 확인합니다.

→

메모가 업로드되나요?

아니요. 계정이나 메모 서버가 없으며 데이터는 PC에만 남습니다.

→

어떤 제한이 있나요?

Windows x64와 로컬 League Client가 필요하고 장치 간 동기화는 지원하지 않습니다.

→

자주 묻는 질문

Riot 계정이 필요한가요?

아니요. PC에 로그인된 League Client가 제공하는 명단만 읽습니다.

다른 사람이 내 메모를 볼 수 있나요?

아니요. 메모는 PC에만 저장됩니다.

Vanguard에 영향을 주나요?

게임에 주입하거나 게임 메모리를 읽지 않습니다.

봇 경기도 기록하나요?

아니요. 봇이 감지되면 경기 전체를 제외합니다.
