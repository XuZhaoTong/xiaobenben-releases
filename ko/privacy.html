개인정보 및 면책

XiaoBenBen의 데이터 처리와 Riot Games 면책 고지입니다.

범위

Windows 앱, Xbox Game Bar 위젯과 공개 웹사이트에 적용됩니다. 시행일: 2026년 7월 10일.

접근 및 저장 데이터

로컬 클라이언트에서 Riot ID, 공개 식별자, 챔피언과 이미지를 읽고 사용자의 표시, 메모와 경기 시간을 PC에 저장합니다.

네트워크

Riot Data Dragon 공개 이미지와 Microsoft Store 또는 GitHub 업데이트에 접근할 수 있으며 광고와 텔레메트리는 없습니다.

공유와 삭제

운영자는 기록을 수집·판매·공유하지 않습니다. 앱 종료 후 %APPDATA%\小本本 폴더를 삭제하면 전체 데이터가 지워집니다.

보안 경계

게임 메모리를 읽거나 코드를 주입하지 않으며 Riot 비밀번호를 요구하지 않습니다.
