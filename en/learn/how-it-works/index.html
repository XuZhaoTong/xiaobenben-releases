How does XiaoBenBen record players?

While running, the app reads the roster exposed by the local League Client and Live Client, excludes you and bots, then stores stable player identifiers together with your notes in a local file.

Updated: 2026-07-10

Data flow

League Client supplies Riot IDs and public identifiers; Live Client can add in-match context; the app merges partial sources and avoids duplicate matches.

Matching old records

It prefers PUUID, summoner ID and a full Riot ID instead of merging players only because their display names match.

Storage

Desktop records stay in the current Windows user's app-data folder. A read-only snapshot is copied to the Game Bar widget in the same package.

Frequently asked questions

What if League Client is closed?

Automatic roster capture is unavailable, but existing notes can still be viewed and edited.

Does it read chat?

No.

What if a player renames?

Stable identifiers can preserve the match; name-only records may require manual confirmation.
