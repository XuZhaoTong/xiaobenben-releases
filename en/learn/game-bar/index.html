How do I review notes with Win+G?

After installing XiaoBenBen, press Win+G and open its widget to see the other nine players, positive and negative counts, and notes from earlier encounters.

Updated: 2026-07-10

Why Game Bar

Game Bar is the Windows game overlay. It avoids injecting into League of Legends and is more reliable over fullscreen games than a normal topmost desktop window.

Steps

Start the desktop app so it can capture the roster; enter a match; press Win+G; open XiaoBenBen from the widget menu; select a player to read history.

Pinning

Pinning is optional. You can open it only when needed and close Game Bar to keep the play area clear.

Frequently asked questions

The widget is missing

Make sure the unified package installed successfully, then look for XiaoBenBen in the Game Bar widget menu.

Can I type a note there?

The current widget focuses on review; full editing is available in the desktop app.

Must I restart the game?

Usually no. Reopen Win+G after first installation.
