Privacy and disclaimer

Data handling, privacy boundaries and the Riot Games disclaimer for XiaoBenBen.

Scope

This notice covers the XiaoBenBen Windows desktop app, Xbox Game Bar widget and public website. Effective July 10, 2026.

Data accessed and stored

The app reads Riot IDs, public player identifiers, champions and profile images from the local League Client / Live Client APIs. Your marks, notes and match times are stored on your own PC.

Network access

The app may fetch public images from Riot Data Dragon. Store builds update through Microsoft Store; direct builds check GitHub Releases. XiaoBenBen contains no ads, telemetry or behavioral analytics.

Sharing, sale and deletion

The operator does not collect, sell or share player notes. Delete individual records in the app, or remove %APPDATA%\小本本 after closing the app to erase all local data.

Security boundary

XiaoBenBen does not read game memory, inject code into the game, or request Riot credentials.

Support

Questions and privacy requests can be filed through GitHub Issues. GitHub Issues
