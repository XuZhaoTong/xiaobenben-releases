能力與界線

它會做

  • 記錄真人對局中除自己外的玩家
  • 儲存 Riot ID、對局時間和自己的評價
  • 遇到有記錄的玩家時發送開局提醒
  • 同步本機 Game Bar 小工具狀態

它不會做

  • 不分析勝率或預測輸贏
  • 不建立公開玩家評分
  • 不讀取記憶體或注入遊戲
  • 不記錄電腦對戰

常見問題指南

安裝、運作方式、隱私與相容性的直接答案。

小本本是什麼？

小本本是 Windows 本機玩家筆記工具，用來記住真人對局中遇到的玩家和自己的評價。

→

小本本如何記錄玩家？

應用讀取本機 League Client 與 Live Client 名單，排除自己和機器人，並把識別與筆記存成本機資料。

→

如何用 Win+G 查看記錄？

按 Win+G 開啟小本本小工具，就能查看本局玩家、正負評價與過去筆記。

→

玩家記錄會上傳嗎？

不會。小本本沒有玩家記錄伺服器或帳號，資料只留在 Windows 電腦。

→

小本本有哪些限制？

只支援 Windows x64，依賴本機 League Client；資料不跨裝置同步，系統通知也可能被全螢幕隱藏。

→

常見問題

需要 Riot 帳號嗎？

不需要，只讀取本機已登入的 League Client 名單。

其他玩家看得到我的評價嗎？

看不到，記錄不會上傳。

會影響 Vanguard 嗎？

不注入遊戲，也不讀取遊戲記憶體。

會記錄電腦對戰嗎？

不會。偵測到機器人後整局不寫入記錄。
