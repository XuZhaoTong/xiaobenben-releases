隱私與免責聲明

小本本的資料界線、隱私說明與 Riot Games 免責聲明。

適用範圍

適用於 Windows 桌面應用、Xbox Game Bar 小工具與公開網站。生效日期：2026 年 7 月 10 日。

存取與保存的資料

應用從本機 League Client / Live Client 讀取 Riot ID、公開玩家識別、英雄與頭像，並儲存使用者自己的評價與對局時間。

網路存取

可能從 Riot Data Dragon 取得公開圖片，並透過 Microsoft Store 或 GitHub 檢查更新；不含廣告或遙測。

分享與刪除

營運者不收集、販售或分享玩家記錄。刪除 %APPDATA%\小本本 可清除全部本機資料。

安全界線

不讀取遊戲記憶體、不注入遊戲，也不要求 Riot 帳號密碼。

支援

問題與隱私請求可提交至 GitHub Issues。 GitHub Issues
