隐私与免责声明

适用范围

本说明适用于小本本 Windows 桌面应用、Xbox Game Bar 小组件及其下载页面。生效日期：2026 年 7 月 10 日。

访问和保存的数据

小本本会从本机 League Client / Live Client 接口读取当前对局玩家的 Riot ID、公开玩家标识、英雄与头像信息，并保存用户自己填写的夯、拉、评价和对局时间。这些记录保存在用户电脑，不会上传到小本本运营者的服务器。

网络访问

应用会访问本机 League Client / Live Client 接口，并可能从 Riot Games Data Dragon 获取公开的英雄与头像资源。Microsoft Store 版本由 Microsoft Store 检查和安装更新；GitHub 直接下载版本会访问 GitHub Release 检查更新。小本本不包含广告、遥测或行为分析服务。

共享与出售

小本本运营者不收集、出售或向第三方共享玩家记录。向 Riot Games Data Dragon、Microsoft Store 或 GitHub 发起网络请求时，相应服务可能按照其自己的隐私政策处理 IP 地址和基础请求日志。

删除数据

用户可以在应用中删除单条玩家记录。若要删除全部本地数据，可以退出小本本后删除 %APPDATA%\小本本 目录。卸载应用不会上传或保留任何服务器端副本。

安全边界

小本本不读取游戏内存，不向游戏进程注入代码，也不要求 Riot Games 账号或密码。

支持与联系

问题和隐私请求可以提交到 GitHub Issues。
