隐私与免责声明

小本本的数据边界、隐私说明与 Riot Games 免责声明。

适用范围

本说明适用于小本本 Windows 桌面应用、Xbox Game Bar 小组件及其公开网站。生效日期：2026 年 7 月 10 日。

访问和保存的数据

小本本从本机 League Client / Live Client 接口读取本局玩家 Riot ID、公开玩家标识、英雄和头像信息，并保存用户填写的夯、拉、评价和对局时间。记录保存在用户自己的电脑。

网络访问

应用可能从 Riot Games Data Dragon 获取公开英雄和头像资源。Microsoft Store 版本由 Microsoft Store 更新；直接下载版会访问 GitHub Release 检查更新。小本本不包含广告、遥测或行为分析。

共享、出售与删除

运营者不收集、出售或共享玩家记录。用户可以删除单条记录；退出应用后删除 %APPDATA%\小本本 可清除全部本地数据。

安全边界

小本本不读取游戏内存、不向游戏进程注入代码，也不要求 Riot Games 账号或密码。
