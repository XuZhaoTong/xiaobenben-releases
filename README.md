# 小本本

英雄联盟本地玩家记录工具。自动记录每局遇到的玩家，下次再遇到时可以通过 `Win+G` 查看以前写过的评价。

## 下载

[下载页](https://xiaobenben.vercel.app) · [下载 Windows 版](https://github.com/XuZhaoTong/xiaobenben-releases/releases/latest/download/xiaobenben_0.4.3_x64-setup.exe) · [反馈问题](https://github.com/XuZhaoTong/xiaobenben-releases/issues)

支持 Windows 10 / 11 x64。一个安装包同时安装桌面端和 Xbox Game Bar 小组件，所有记录只保存在本机。

> 安装包目前没有商业代码签名证书，Windows SmartScreen 可能显示“未知发布者”。
